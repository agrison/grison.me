Peculium, simple web app to manage your savings.

Peculium which means savings in Latin, is a simple Web application (work in progress) that can be used locally on your computer to manage personal savings. It is an easy and simple alternative to full applications like GNU Cash as it does handle just simple stupid things.

  • Handle banking accounts
  • Graphics and statistics about monthly and yearly savings.
  • Monthly expenses details

This applications was developed for myself, firstly to replace my Excel document and like always, in order to learn new stuff like AngularJS, DropWizard and MongoDB.

Installation

  1. You need at least Java 5 to run the application, it is pretty standard stuff so you may already have it installed on your system.
  2. Download the JAR file on my Github project page
  3. Put it anywhere you want on your system.

Running the application

The application is bundled as a simple JAR file containing the whole needed dependencies, to simplify the installation process. Just run the peculium.jar file, a local web server will be launched on your localhost, and your default browser opened on it. That's it, you're ready to use peculium.

Configuring the application

At first launch peculium will show you the configuration screen, so that you will be able to configure

  • The persons that are actually saving money (you, your husband/wife, or your genius dog)
  • The bank accounts (name, rate, max deposit) of these persons
  • The currency

Where are saved my datas?

The datas are saved in a locally installed H2DB database file. This file named peculium.h2db is located in the same directory as the peculium.jar file.
